ample

A general implementation of Approximate Message Passing (AMP) for Matlab.

Note: This repository is still under active development. There may be some hiccups.

Intro

The goal of this problem is to provide a quick-to-use and easily generalizable implementation of the approximate message passing (AMP) algorithm for calculating factorizations of posterior distributions to solve inverse problems. Specifically, ample is built for solving compressed-sensing (CS) signal reconstruction problems of the form \[ y = A x + \mathcal{N}(0,\Delta I_{M\times M}),\] where \( y \) is of dimensionality \( M\). The original unknown signal \( x\) is of some dimensionality \( N\) and is distributed according to some known prior described by parameters \( \theta \), \( x \sim P_0(x;\theta) \).

  • Quick-to-use: At its core, one only needs to provide the set of obstervations, the system which obtained those observations, and the type of signal prior. However, ample contains a number of options to allow the user to tailor AMP to suit their particular needs.
  • Generalizable: ample provides a framework by which the user can easily implement their own priors to fit AMP to their specific application.
